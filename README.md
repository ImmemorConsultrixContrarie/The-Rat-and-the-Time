# The Rat and The Time

An idle game where The Rat hunts Grass to become stronger.

## Game Mechanics

- **The Rat** starts by killing **1 Grass per minute**
- Each kill grants:
  - **1% faster** killing speed (multiplicative)
  - **0.1 Grass per minute** additional production
- Progress happens in real-time (no offline progress)

## Getting Started

Run the main file:

```bash
node main.js
```
